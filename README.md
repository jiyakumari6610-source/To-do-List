# 📋 Task Builder App

A simple command-line task management application built with Python that allows users to add, view, and manage their daily tasks efficiently.

## 🚀 Features

- **Add Tasks** ➕ – Quickly add new tasks to your to-do list
- **View Tasks** 👀 – Display all your pending tasks with numbered indices
- **Exit** 🚪 – Clean exit from the application
- **User-Friendly Interface** – Simple menu-driven navigation with emoji feedback
- **Error Handling** – Validates user input and handles empty task lists gracefully
